## 二、演示视频（3-5 分钟）

### 录制工具
- Windows：`Win + G` 打开 Game Bar，点"录制"
- 或下载 OBS Studio（免费）

### 录制脚本

| 时间 | 画面 | 操作 |
|------|------|------|
| 0:00-0:20 | 终端 | 跑 `python main.py`，展示四场景对比表 |
| 0:20-0:50 | 浏览器 | 打开 `http://127.0.0.1:5000`，登录 Alice / alice123，展示验证码刷新、顶栏余额高亮 |
| 0:50-1:30 | Swap 页面 | 输入 1 ETH → 查看实时报价（汇率/滑点/手续费）→ 注意右侧储备构成图同步更新 → 点击交换 |
| 1:30-2:10 | 流动性页面 | 输入 ETH 数量 → 展示 USDC 自动配对计算 → 添加流动性；切换到移除 Tab 展示移除功能 |
| 2:10-2:40 | 仿真页面 | 下拉选"大额冲击场景"→ 点"运行完整仿真"→ 展示价格变化图和事件日志 |
| 2:40-3:00 | 盈亏分析 | 展示无常损失曲线，HODL vs LP 柱状对比图 |
| 3:00-3:20 | 仪表盘 | 展示价格走势 + 深度曲线 + 交易记录（Swap + 流动性事件合并显示） |
| 3:20-3:40 | 退出 | 点退出 → 弹出确认框 → 确认 → 回到登录页 |
